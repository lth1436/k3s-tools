라즈베리파이의 **마스터 노드**에서 다음을 실행하세요:

```bash

```

```bash
cd ~/k3s-monitor

./cluster-diagnostics.sh # 상세 진단
```

curl -O https://raw.githubusercontent.com/your-username/k3s-tools/main/quick-status.sh && chmod +x quick-status.sh

curl -O https://raw.githubusercontent.com/your-username/k3s-tools/main/realtime-monitor.sh && chmod +x realtime-monitor.sh

# 3. 권한 수정 (필요시)
sudo bash fix-k3s-permissions.sh

## 🆘 문제 해결

### "curl: command not found"

```bash
# curl 설치
sudo apt update
sudo apt install curl -y

# 또는 wget 사용
wget -O - https://raw.githubusercontent.com/YOUR-USERNAME/k3s-tools/main/install.sh | bash
```

### "Permission denied" (권한 에러)

```bash
# 자동 권한 수정
sudo bash ~/k3s-monitor/fix-k3s-permissions.sh
```

### "Connection refused" (연결 거부)

```bash
# GitHub 접근 문제일 수 있음
# 1. 인터넷 연결 확인
ping github.com

# 2. DNS 확인
nslookup github.com

# 3. 방화벽 확인
sudo ufw status

# 4. 로컬 저장소 사용
# (마스터 노드에서 HTTP 서버 시작)
python3 -m http.server 8000

# (다른 노드에서)
curl http://master-ip:8000/install.sh | bash
```

### "urllib.error" (다운로드 스크립트 오류)

```bash
# Python3 설치 확인
python3 --version

# 없으면 설치
sudo apt install python3 -y
```
